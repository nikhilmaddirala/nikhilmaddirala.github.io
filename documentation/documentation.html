Summit Theme Documentation

Responsive vCard template by @dahndesigns.

1. Summit theme

1.1 Introduction

Thanks a lot for purchasing my theme! I really appreciate it! If you have a question and you can't find the answer in this help file, feel free to contact me through my ThemeForest profile. Thanks again!

Summit is a premium, clean and responsive vCard theme. It is great for showing who you are, showing your work and to allow people to get in touch with you. It looks awesome on phones and tablets as well!

1.2 The package

Download the file from your ThemeForest downloads. Inside you'll find the following files and folders:

  • HTML folder
  • Documentation folder
  • PSD folder

2. The head

2.1 Opening the file

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and find the Head section. It'll look like this:

<!doctype html>
<head>
  <meta charset="utf-8">
  <title>Your title</title>
  <meta name="description" content="Description for your site">
  <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=yes" />
  
  <link rel="stylesheet" href="css/style.css">
  <link rel="stylesheet" href="css/grid.css">
  <link rel="stylesheet" href="css/elements.css">
  <link rel="stylesheet" href="css/prettyPhoto.css">
</head>
2.2 Changing the title and description

The only thing you'll want to change here is the title and the description. Find <title>Your title</title> and replace it with your own page title. Next, find <meta name="description" content="Description for your site"> and add your own description.

3. Header

3.1 Opening the file

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and scroll to the Header section. It'll look like this:

  <!-- HEADER -->
  <div class="header">
    <div class="logo"><h2>FIRST<strong>LAST</strong></h2></div>
    <div class="menubutton"></div>

    <div class="menu">
      <ul>
        <li id="default"><a href="#homepage">Home</a></li>
        <li><a href="#portfolio">Portfolio</a></li>
        <li><a href="#blog">Blog</a></li>
        <li><a href="#resume">Resume</a></li>
        <li><a href="#contact">Contact</a></li>
      </ul>
    </div>

    <br class="clear" />
  </div>
3.2 Changing the name

The only thing you'll want to change here is your name. Find <h2>FIRST<strong>LAST</strong></h2> and replace "FIRST" with your first name and "LAST" with your last name. That's all!

4. Home page

4.1 Opening the file

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and scroll to the Homepage section. It'll look like this:

<!-- HOMEPAGE -->
  <div class="wrapper homepage" id="homepage">
    <div class="container">
      <div class="three columns">       
        <div class="frame">
          <img src="img/yourphoto.jpg">
        </div>
      </div>
      
      <div class="five columns">
        <h1>YOUR JOB/SLOGAN</h1>
        <p><strong>Bold intro text</strong></p>
        <p>More intro text</p>
      </div>
      
      <div class="four columns">
        <div class="left-divider">
          <p>Text above your social buttons<p>       
          <a href="link to your twitter"><button class="twitter">@yourname</button></a>
          <a href="link to your facebook"><button class="facebook">Your name</button></a>
          <a href="link to your linkedin"><button class="linkedin">Your Name</button></a>
          <a href="mailto:you@youremail.com"><button class="mail">you@youremail.com</button></a>
        </div>
      </div>

   </div>
  </div>
4.2 Adding your photo

Adding your photo is very easy. First of all, add your photo to the img folder. Make sure to remember what name you gave it. Next, find <img src="img/yourphoto.jpg"> in the code. Replace yourphoto.jpg with the name you gave your own photo and you're done!

4.3 Adding your personal info

The next step is to add your personal info. Find <h1>YOUR JOB/SLOGAN</h1> in the code and replace "YOUR JOB/SLOGAN" with your own job or slogan. The next two lines allow you to tell some more about yourself. Simply replace "Bold intro text" and "More intro text" with your own text.

4.4 Adding your social networks

Now it's time to add your social networks. Replace the links after the "href=" with the actual links. Now add your actual name and your homepage is done!

5. Portfolio

5.1 Opening the file

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and scroll to the Portfolio section. It'll look like this:

  <!-- PORTFOLIO -->
  <div class="wrapper portfolio" id="portfolio">
    <span class="big-text">portfolio</span>
    <div class="container">
      <div class="twelve columns slogan">
        <h5>Your tags/motto</h5>
      </div>

      <div class="items">      
        <div class="twelve columns">
          
          <a href="img/bigimage.jpg" rel="prettyPhoto" title="Image caption here">
            <div class="item">
              <img src="img/smallimage.jpg">
                <div class="mask">
                <h6>Title on hover</h6>
                <p>Description on hover</p>
                <span class="tags">tags</span>
                </div>
            </div>
          </a>

        </div>
      </div>

   </div>
  </div>
5.2 Adding your tags/motto/slogan

To change the text that you see before your items, simply find <h5>Your tags/motto</h5> and replace "Your tags/motto" with whatever you want.

5.3 Adding portfolio items

You'll probably want to add some of your work as well. No problem! This is the structure of a portfolio item:

          <a href="img/bigimage.jpg" rel="prettyPhoto" title="Image caption here">
            <div class="item">
              <img src="img/smallimage.jpg">
                <div class="mask">
                <h6>Title on hover</h6>
                <p>Description on hover</p>
                <span class="tags">tags</span>
                </div>
            </div>
          </a>

I think most of it is pretty much self-explanatory. Change bigimage.jpg to the name of the image you want to show (make sure it's located inside the img folder). On the same line, find title="Image caption here" and add your image caption.

When hovering over the small preview of your item, the visitor sees a little description. First of all, change smallimage.jpg to the name of the small image preview (I recommend a width of 300 pixels. Also, it looks best when every small image has the same height). Now add your title, description and tags and you're done!

You can add as much portfolio items as you like.

6. Blog

6.1 Opening the file

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and scroll to the Blog section. It'll look like this:

  <!-- BLOG -->
  <div class="wrapper blog" id="blog">
    <span class="big-text">blog</span>
    <div class="container">
      
        <div class="ten columns offset-by-two post-title">
          <h5>Post title</h5>
        </div>
        <div class="two columns info">       
          <span class="date">date</span>
          <span class="author">author</span>
          <span class="tags">tag</span>
        </div>
        <div class="ten columns post">
          <img src="img/blogpostimage.jpg">
          <p><strong>Bold text</strong></p>
          <p>More text</p>
          <a href="#_" class="readmore">Read more &rarr;</a>
          <div class="more">
            <p>Text after read more button</p>
          </div>
        </div>

   </div>
  </div>
6.2 Adding a post

You might want to add some blog posts. This is the markup for a post:

        <div class="ten columns offset-by-two post-title">
          <h5>Post title</h5>
        </div>
        <div class="two columns info">       
          <span class="date">date</span>
          <span class="author">author</span>
          <span class="tags">tag</span>
        </div>
        <div class="ten columns post">
          <img src="img/blogpostimage.jpg">
          <p><strong>Bold text</strong></p>
          <p>More text</p>
          <a href="#_" class="readmore">Read more &rarr;</a>
          <div class="more">
            <p>Text after read more button</p>
          </div>
        </div>

Simply replace the text, title and info with your own. Replace blogpostimage.jpg with the image that belongs to the post (I recommend a width of 580 pixels). If you want to add another paragraph, simply place your text between <p> tags, like this: <p>Your paragraph</p>.

You can add as much posts as you like. If you want to place a divider between two posts, add this in between the posts:

<div class="ten columns offset-by-two"><div class="divider"></div></div>

If you want to add a quote, use this. You can add class="right" to align it to the right.

<blockquote>The quote<span>- Who said it</span></blockquote>

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and scroll to the Resume section. It'll look like this:

  <!-- RESUME -->
  <div class="wrapper resume" id="resume">
    <span class="big-text">resume</span>
    <div class="container">
      
      <div class="eight columns">
        
        <div class="education">
          <h5>Education</h5>

          <div class="date">date</div>
          <div class="info">
            <div class="heading">
              <h6>education title</h6>
              <span class="place">location</span>
            </div>
            <p class="description">education description</p>
          </div>

        </div>
 
        <div class="experience">
          <h5>Experience</h5>

          <div class="date">date</div>
          <div class="info">
            <div class="heading">
              <h6>Your job/function</h6>
              <span class="place">company</span>
            </div>
            <p class="description">job description</p>
          </div>

        </div>    

      </div>

      <div class="four columns">

          <div class="skills">

          </div>

          <a href="link to your resume"><button class="big">Download my resume</button></a>
        </div>
      </div>
    </div>
  </div>
7.2 Adding your education

The first thing you'll want to add is your education. This is what it looks like:

          <div class="date">date</div>
          <div class="info">
            <div class="heading">
              <h6>education title</h6>
              <span class="place">location</span>
            </div>
            <p class="description">education description</p>
          </div>

Simply replace the text, date, description etc with your own. You can add as much of them as you like.

7.3 Adding your experience

The next thing you'll want to add is your experience. This is what it looks like:

          <div class="date">date</div>
          <div class="info">
            <div class="heading">
              <h6>Your job/function</h6>
              <span class="place">company</span>
            </div>
            <p class="description">job description</p>
          </div>

Pretty much the same as Education. Simply replace the text, date, description etc with your own. You can add as much of them as you like.

7.4 Adding your skills

You might want to show your skills. This is how you do it:

            <div class="skill-container">
              <div class="skill" style="width: 65%;">Your skill (change percentage to the left)</div>
            </div>

Change "Your skill" to your actual skill. Change the percentage to how good you are at that skill (eg: if you're pretty good at something, you could change it to 80%). To add another skill, simply add another skill-container.

7.5 Adding a resume button

The last thing to add is a link to download your resume:

<a href="link to your resume"><button class="big">Download my resume</button></a>

Change "link to your resume" to the link to your actual resume (eg: if your resume is named resume.pdf, place it in the same folder as index.html and link to resume.pdf). And that's it, your resume is done!

8. Contact

8.1 Opening the file

Open up the index.html file in the HTML folder using a text editor like Notepad (not Word), and scroll to the Contact section. It'll look like this:

  <!-- CONTACT -->
  <div class="wrapper contact" id="contact">
    <span class="big-text">contact</span>
    <div class="container">
      <div class="twelve columns">
        <div class="map"><img src="img/yourcontactimage.jpg"></div>
      </div>
      <div class="four columns">
        <h5>Contact info</h5>
        <ul>
          <li><strong>Name:</strong> Your name</li>
          <li><strong>Address:</strong> Your address</li>
          <li><strong>E-mail:</strong> you@youremail</li>
          <li><strong>Phone:</strong> Your phone number</li>
        </ul>

      </div>
      <div class="eight columns">
      <form method="post" action="send.php" id="contactform">
        <label for="name">Name:</label>
        <input type="text" name="name" id="name" class="required" />
 
        <label for="email">Email:</label>
        <input type="text" name="email" id="email" class="required email"/>
        
        <label for="message">Message:</label>
        <textarea name="message" id="message" class="required"></textarea>

        <button type="submit" name="submit" value="Submit" class="submit-button">Send message</button>
      </form>
      </div>

    </div>
  </div>
8.2 Adding an image

You can add an image above the contact form. Simply add it to the img folder and change "yourcontactimage.jpg" to the correct name. I recommend a width of 680 pixels.

8.3 Adding your personal info
        <ul>
          <li><strong>Name:</strong> Your name</li>
          <li><strong>Address:</strong> Your address</li>
          <li><strong>E-mail:</strong> you@youremail</li>
          <li><strong>Phone:</strong> Your phone number</li>
        </ul>

Simply replace the info with your own. Feel free to add or remove certain lines.

8.4 Configuring the contact form

Much easier than it sounds like. Simply open send.php in the HTML folder and on the third line, replace the e-mail address with your own. That's all, the theme takes care of the rest. Please note that it only works on an actual server.

9. Footer

9.1 Opening the file
